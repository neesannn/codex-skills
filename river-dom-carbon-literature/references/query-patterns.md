# Query Patterns

Use these as starting points, then adjust for the user's exact scope and date window.

## EEM-PARAFAC River DOM

- `"EEM-PARAFAC" river dissolved organic matter 2026`
- `"EEM-PARAFAC" riverine DOM Water Research`
- `"PARAFAC" "riverine dissolved organic matter" "Journal of Hydrology"`
- `"excitation-emission matrix" "river" "dissolved organic matter"`
- `"EEM-PARAFAC" "urban river" DOM`
